# Escola de Aviacao - Sistema de Gestao

Projeto Java multi-modulo para gerir uma escola de aviacao.

## Modulos

- `shared`: entidades e DAL
- `backend-common`: servicos de negocio
- `desktop`: aplicacao Swing
- `web`: API Spring Boot
